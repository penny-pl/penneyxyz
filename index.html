转换公式

笛卡尔 → 球坐标:

r = √(x² + y² + z²)

θ = arccos(z/r)

φ = arctan(y/x)

球坐标 → 笛卡尔:

x = r·sin(θ)·cos(φ)

y = r·sin(θ)·sin(φ)

z = r·cos(θ)

注：θ (极角) 是与 Z 轴的夹角，φ (方位角) 是在 XY 平面上与 X 轴的夹角

φ (方位角/Azimuth)
